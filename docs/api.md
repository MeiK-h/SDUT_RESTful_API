# API 文档

## 电量查询接口（不需要登录）

**URL**

`/public/energy`

**请求类型**

`POST`

**请求参数**

| 参数 | 类型 | 含义 |
| - | - | - |
| `floor` | string | 公寓号（可用列表见下方[公寓列表](#公寓列表)） |
| `room` | string | 房间号 |

**请求示例**

```bash
curl http://<url>/public/energy --request POST --form 'floor=01#南' --form 'room=101'
```
